## workbenchScalajs
 Simple project for working with scalajs. 
 Run:

```./scripts/workbench-scalajs.sh```

 ... [open a browser](http://localhost:12345/modules/workbenchScalajs/target/scala-2.11/classes/index-dev.html) and play around.
